
# 🌐 Multilingual Wrapper (GitHub Pages Ready)

Legg inn Rise 360-eksportene i mappene under `lang/`.
Mapper er fylt med **placeholder index.html** slik at de vises på GitHub.
Bytt ut placeholder-filene med ekte `index.html` + assets fra hver språkeksport.

Struktur:
- index.html (språkvelger)
- lang/no/  ← Norsk (erstatt placeholderen med Rise-filer)
- lang/en/  ← Engelsk
- lang/pl/  ← Polsk
